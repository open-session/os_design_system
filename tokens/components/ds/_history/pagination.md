# Pagination Primitive — Migration History

## Phase C Wave 5 — UUI v8 alignment (2026-05-08)

Pulled the full Pagination family for v8: `pagination`, `pagination-base`, `pagination-dot`, `pagination-line`. Rescoped from C4 to C5 because v8's new `PaginationCardAdvanced` embeds `Select` + `InputBase` and Pagination needed the Select family pulled first.

### Type 2 absorbed wholesale

- **`PaginationCardAdvanced` (new component).** v8 introduces a richer pagination variant with: an editable page-number input (`InputBase`), a "Rows per page" `Select` dropdown, double-chevron jump-to-first/jump-to-last buttons (`ChevronLeftDouble`, `ChevronRightDouble`), and a mobile fallback ("Page X of Y" condensed). Adopted greenfield.
- **`pageSize` + `onPageSizeChange` props** added to `PaginationCardMinimal`. v8 augments — does NOT replace — the existing "Page X of Y" string with a sibling page-size `Select`. The Type 3 question of "BOS retains 'Page X of Y' as a wrapper" was moot — v8 itself retains it.
- **Item sizing collapsed `size-10` → `size-9`** across the family (PaginationItem, Pagination.Ellipsis in PageDefault/PageMinimalCenter/CardDefault, ButtonGroupItem in PaginationButtonGroup).
- **`PaginationButtonGroup` size collapsed `md` → `sm`** to align with the new family-wide compact spacing.
- **Motion easing relaxed.** `ease-motion-out` → `ease-linear` on PaginationItem hover transition. Same Toggle Wave 2C / Dropdown C4 / Select Wave C5a precedent.
- **Pagination.Ellipsis spread fix.** `pagination-dot.tsx` and `pagination-line.tsx` now spread `{...page}` on the Ellipsis branch (was missing) — accessibility / data-attribute wiring fix.

### Type 3 retained (hand-fix on re-pull)

- **`Button` import path.** v8 sidecar imports from `@/components/base/base/buttons/button` (vendor primitive). BOS retains import from `@/components/ds/buttons/button` (Shape-C wrapper at `components/ds/buttons/button.tsx`, preserves the calm/contained primary CTA per Decision #2). Hand-restored after sidecar copy. Same recurring Type 3 pattern as Pro icon retention — file under "Type 3 retained manually, hand-fix on re-pull."
