# Brand Identity

This document defines the visual identity system for Open Session, including logo usage, color palette, and typography standards.

---

## Logo System

Open Session uses a comprehensive logo system with multiple variations for different contexts and applications.

### Main Logo (Combination Mark)

The main logo combines the brandmark (globe icon) with the stacked wordmark "open session."

#### Spacing

The preferred clear space around the logo is the height of the Open Session letters. This ensures the logo maintains visual impact and legibility in all applications.

#### Minimum Usage

| Format  | Minimum Size |
|---------|--------------|
| Print   | 1.5"         |
| Digital | 110px        |

---

### Wordmark Stacked

The stacked wordmark displays "open" above "session" without the brandmark icon.

#### Spacing

The preferred clear space around the stacked wordmark is the height of the Open Session letters.

#### Minimum Usage

| Format  | Minimum Size |
|---------|--------------|
| Print   | 1"           |
| Digital | 70px         |

---

### Wordmark Horizontal

The horizontal wordmark displays "open session" on a single line.

#### Spacing

The preferred clear space around the horizontal wordmark is the height of the Open Session letters.

#### Minimum Usage

| Format  | Minimum Size |
|---------|--------------|
| Print   | 1.5"         |
| Digital | 110px        |

---

### Brandmark

The brandmark is the standalone globe icon that can be used independently of the wordmark.

#### Spacing

The preferred clear space around the brandmark is the width of the inner star.

#### Minimum Usage

| Format  | Minimum Size |

| Print   | 0.5"         |
| Digital | 50px         |

---

### Monogram

The monogram "OS" provides an abbreviated logo option for constrained spaces.

#### Spacing

The preferred clear space around the logo is the height of the Open Session letters.

#### Minimum Usage

| Format  | Minimum Size |
|---------|--------------|
| Print   | 1.5"         |
| Digital | 110px        |

## Logo Variations

All logo variations are available in three color treatments:

| Color Treatment | Description                                      |
|-----------------|--------------------------------------------------|
| **Vanilla**     | Light cream color on dark backgrounds            |
| **Charcoal**    | Dark color on light backgrounds                  |
| **Glass**       | Translucent/decorative effect for secondary use  |

### Logo Types Matrix

| Variation             | Vanilla | Charcoal | Glass |
|-----------------------|---------|----------|-------|
| Combination           | Yes     | Yes      | Yes   |
| Wordmark Stacked      | Yes     | Yes      | Yes   |
| Wordmark Horizontal   | Yes     | Yes      | Yes   |
| Brandmark             | Yes     | Yes      | Yes   |
| Monogram              | Yes     | Yes      | Yes   |
| Secondary Icons       | Yes     | Yes      | Yes   |

**Secondary Icons:** Four-point star icons (outline and filled variations) used as accessory elements.

---

## Logo Application Guidelines

### Do's

- **Do** use the combination logo as the default logo wherever possible
- **Do** use the brandmark logo for profile images and favicons
- **Do** use the horizontal wordmark for display areas with minimal vertical space
- **Do** use the monogram logo as a secondary option for profile images and favicons
- **Do** use secondary icons as accessory elements
- **Do** use glass effect logos as secondary, decorative branding

### Don'ts

- **Don't** use the combination logo in small display areas
- **Don't** use the brandmark logo mixed with any other logo variations
- **Don't** use the horizontal wordmark for profile images or favicons
- **Don't** use the monogram logo mixed with any other logo variations
- **Don't** use secondary icons as the sole representation of our brand or mixed with any other logo variations
- **Don't** use glass effect logos when clear branding is required

## Color Application Guidelines

### Do's

- **Do** use Vanilla or Charcoal as a background fill color
- **Do** use Vanilla color filled logos and copy on Charcoal color filled backgrounds, and Charcoal color filled logos and copy on Vanilla color filled backgrounds
- **Do** use Aperol color filled elements on both Vanilla and Charcoal filled backgrounds
- **Do** use Aperol color fill for numbers on a numbered list or subheading copy on a Charcoal color filled background

### Don'ts

- **Don't** use Aperol as a background fill color
- **Don't** use Aperol color filled logos on Vanilla color filled background, only on Charcoal color filled backgrounds
- **Don't** use Aperol color fill for body copy or large title copy
- **Don't** use Aperol color filled copy on Vanilla color filled backgrounds

## Typography

### Font Families

Open Session uses two primary typefaces:

1. **Neue Haas Grotesk Display Pro** - For headings and display text
2. **Neue Haas Grotesk Text Pro** - For body text and UI elements
3. **OffBit** - For subheadings (accent typeface)

### Type Scale

| Use Case                  | Typeface                        | Weight     |
|---------------------------|---------------------------------|------------|
| Presentation              | Neue Haas Grotesk Display Pro   | 75 Bold    |
| Display & Large Headings  | Neue Haas Grotesk Display Pro   | 75 Bold    |
| Small Headings            | Neue Haas Grotesk Display Pro   | 65 Medium  |
| Subheadings               | OffBit                          | 101        |
| Body                      | Neue Haas Grotesk Text Pro      | 55 Roman   |
| Button                    | Neue Haas Grotesk Text Pro      | 65 Medium  |
| Label                     | Neue Haas Grotesk Text Pro      | 65 Medium  |
| Caption                   | Neue Haas Grotesk Text Pro      | 55 Roman   |

### Font Weight Reference

| Weight Name | Numeric Value |
|-------------|---------------|
| 55 Roman    | Regular (400) |
| 65 Medium   | Medium (500)  |
| 75 Bold     | Bold (700)    |
| 101         | OffBit weight |
